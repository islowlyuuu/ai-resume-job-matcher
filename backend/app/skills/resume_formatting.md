# Resume Formatting Skill

目标：让 AI 优化后的简历内容更适合继续编辑、导出和投递，而不是只生成一段散文。

排版原则：

- 内容应按“标题 / 摘要 / 经历要点 / 技能关键词 / 开场白”组织，方便导出到 Word 或 PDF。
- `optimized_summary` 应适合作为简历顶部摘要，避免太长，通常 80-160 个中文字符。
- `rewritten_bullets` 每条控制在 35-90 个中文字符左右，便于 Word 中换行和阅读。
- bullet 之间信息不要重复：一条突出业务场景，一条突出技术实现，一条突出结果或边界处理。
- `ats_keywords` 应是可以自然放入简历的关键词，不要混入太长的句子。
- 避免在导出内容中加入 Markdown 表格、复杂编号或过多 emoji。

美观要求：

- 标题清晰，不堆叠过多技术名词。
- 摘要可读，不要像模型生成的自我评价。
- 经历 bullet 尽量长短接近，方便导出后形成稳定版面。
- 如果候选人经历较弱，优先通过结构化表达提升专业感，不要夸大事实。
